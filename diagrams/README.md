# Diagrams

This directory stores supporting architecture and analysis diagrams for the
SchoolMaster platform documentation.

## Subdirectories

- `erd/`: entity relationship diagrams
- `use-cases/`: user and system interaction diagrams
- `sequence/`: request and workflow sequence diagrams
- `flowcharts/`: process flow diagrams
